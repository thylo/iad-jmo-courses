## Qu'est-ce que Touch for Luck ?

**Touch for Luck** est la première commission digitale du **M+ Museum** (musée d'art visuel contemporain de Hong Kong), créée par le studio de design interactif **Moniker** (Luna Maurer et Roel Wouters). Lancée le 27 janvier 2022, cette installation de média public transforme le port de Hong Kong en champ de jeu collectif invisible relié par des téléphones mobiles.

## Le Concept : L'Architecture Tactile

Touch for Luck opère une magnification radicale du geste le plus intime de l'ère numérique—le toucher de l'écran mobile—en le transformant en spectacle urbain monumental :

- **Chaque joueur** : Touche son écran sans arrêt pour faire avancer son avatar-poisson
- **Chaque poisson** : Agrandi 1000x et projeté sur la **M+ Facade** (7000 m², 65m x 110m)
- **L'étang partagé** : 1000s de poissons coexistent simultanément, contrôlés par joueurs partout dans le monde
- **La récompense** : Chaque moment de persistance génère des "lucky charms" (gemmes) decorant le poisson
- **L'ironie** : Plus on joue addictif, plus belles deviennent les créations

## Mécanique Interactive

### Le Geste Fondamental
Il n'y a qu'UN geste possible : **toucher l'écran sans relâcher**. Pas de navigation, pas de boutons, pas de menus. Juste la pression continue du doigt transformée en présence persistante.

```
Écran Mobile (intime)      →   M+ Facade (public)
Mon doigt (micro)          →   Poisson géant (macro)
15cm x 8cm                 →   65m x 110m
```

### Structure de Jeu
1. **Accès** : Joueurs se connectent via `touchforluck.com` sur smartphone
2. **Présence** : Chaque doigt relâché = perte d'un "Un-Touch-Time" (buffer de temps)
3. **Récompenses** : Lucky charms accumulés = visuels de poisson plus richement décorés
4. **Visibilité** : Son propre poisson 1000x agrandi sur la façade, visible jusqu'à 1.5km
5. **Synchronisation** : Jeu affiché nightly de 12h-15h CET sur la façade

### Symbologie des Lucky Charms
Les gemmes accumulées métaphorisent les mécanismes addictifs des réseaux sociaux :
- Likes (cœurs numériques)
- Followers (croissance du nombre)
- Trending (visibilité maximale)
- Views (mesure d'attention)

## Pourquoi C'est Innovant

### Pour la Critique Média par l'Expérience
Plutôt que de *parler* de l'addiction aux écrans, Touch for Luck la **materialise corporellement**. En jouant, on expérimente littéralement :

- **Compulsion** : Comment le désir de "plus" (plus de charms) crée dépendance
- **Corporéité** : Comment le geste répétitif du doigt génère fatigue physique
- **Visibilité** : Comment la magnification publique du geste crée obligation de persistance
- **Collectif** : Comment les autres joueurs visibles crée émulation compétitive

C'est une **critique phénoménologique** plutôt que discursive.

### Pour l'Intégration Digital-Urbain
Touch for Luck démontre comment technologie mobile **transforme l'architecture urbaine monumentale**. La façade LED n'est pas décoration ; elle est le **site principal de l'œuvre**. Hong Kong devient galerie collective où paysage naturel (Victoria Harbour) fusionne avec paysage technologique.

### Pour la Performance Collective
Contrairement aux jeux multijoueurs traditionnels (compétition, hiérarchie, victoire/défaite), Touch for Luck crée **présences parallèles coégalitaires** :

- Pas de "gagnant" ou "perdant"
- Chaque avatar a valeur égale sur la façade
- Temps réel (synchrone) mais sans coordination scénaristique
- Chorale d'identités numériques sans directeur

### Pour le Design Critique
Studio Moniker crée une **beauté addictive pour critiquer la beauté addictive**. L'œuvre n'est pas "laide" ou "pénible" ; elle est esthétiquement séduisante. Cette contradiction force confrontation émotionnelle avec la critique intellectuelle.

## Pour la Classe

Touch for Luck offre un **cas d'étude pour design critique et culture médiatique**. Recommandé pour :

### Éducation Médiatique Critique
- **Visite et participation** : Classe touche leurs écrans simultanément, observe leurs poissons sur façade
- **Post-jeu** : Débat immédiat sur expérience d'addiction
- **Analyse design** : Comment les charms rendent compulsion esthétique ? Qu'enleverait du design pour le rendre "honnête" ?

### Étude du Design Critique
- Exemples de design utilisant **immersion pour révéler systèmes**
- Tension entre beauté et critique : peut-on être magnifique en critiquant l'addiction à la magnificence ?
- Comparaison : Touch for Luck vs. films/articles *sur* l'addiction (quelle efficacité pédagogique ?)

### Projets Créatifs
- **Créer installation critique locale** : Classe transforme geste urbain local (e.g., marche, graffiti) en critique via magnification/visualisation
- **Analyse sémiotique** : Pourquoi poissons ? Pourquoi Hong Kong ? Pourquoi "luck" ?
- **Réinvention** : Reprendre mécanique de Touch for Luck pour critiquer autre système (surveillance, consommation, etc.)

## Contexte et Postérité

Touch for Luck s'inscrit dans une **trajectoire d'art/design critique explorant technologie**:

### Studio Moniker
Luna Maurer et Roel Wouters sont connus pour interroger "the characteristics of technology, including how we use it and how it influences our daily lives." Touch for Luck est culmination de cette pratique : **design comme critique incarnée**.

### Lignée Critique
- **Précurseurs** : James Bridle (New Dark Age), Cathy O'Neil (Weapons of Math Destruction)
- **Contemporains** : Expositions critiques sur social media (Serpentine, MoMA, Centre Pompidou)
- **Différence** : Touch for Luck exige *participation corporelle* pour comprendre la critique (vs. observation/lecture)

### Postérité Possible
- Modèle pour musées désirant installation interactive critique (vs. jeux sans contention)
- Inspiration pour architectes/villes intégrant technologie critique dans paysage urbain
- Référence pour designers critiques : comment créer beauté addictive pour critiquer addiction ?
