## Ce qui compte vraiment

### 1. Sélection et manipulation du DOM

Savoir sélectionner un élément dans la page et le modifier. Sélectionner par id, classe, ou type. Modifier le contenu texte, le HTML interne, les attributs. Ajouter ou retirer des classes. Créer de nouveaux éléments et les insérer.

Les méthodes essentielles : `querySelector`, `querySelectorAll`, `getElementById`, `addEventListener`, `createElement`, `appendChild`, `remove`, `classList.add`, `classList.remove`, `classList.toggle`.

**La différence entre `textContent` et `innerHTML`**

`textContent` modifie uniquement le texte d'un élément. Si vous écrivez `element.textContent = '<strong>Bonjour</strong>'`, le texte affiché sera littéralement `<strong>Bonjour</strong>`, balises comprises. Le HTML n'est pas interprété.

`innerHTML` modifie le HTML d'un élément. Si vous écrivez `element.innerHTML = '<strong>Bonjour</strong>'`, le texte affiché sera **Bonjour** en gras. Le HTML est interprété et rendu.

Utilisez `textContent` quand vous voulez afficher du texte simple. Utilisez `innerHTML` quand vous avez besoin d'ajouter des balises HTML. Mais attention : `innerHTML` peut poser des problèmes de sécurité si vous y injectez du contenu venant de l'utilisateur sans vérification.

### 2. Styling via JavaScript

Modifier l'apparence d'un élément depuis JavaScript. Soit en ajoutant ou retirant des classes CSS, soit en modifiant directement les styles inline.

Privilégier les classes CSS pour les changements d'état : `element.classList.add('active')` plutôt que `element.style.color = 'red'`. Les styles inline pour des valeurs dynamiques : positions calculées, tailles variables, couleurs générées.

Les propriétés CSS en JavaScript utilisent la camelCase : `backgroundColor` au lieu de `background-color`, `fontSize` au lieu de `font-size`.

### 3. Animations et temporalité

Créer des animations simples avec trois outils différents.

`setTimeout` exécute une fonction une seule fois après un délai. Utile pour déclencher quelque chose plus tard.

`setInterval` exécute une fonction de manière répétée à intervalle régulier. Attention : il faut stopper l'intervalle avec `clearInterval` sinon il tourne indéfiniment.

`requestAnimationFrame` exécute une fonction avant le prochain rafraîchissement de l'écran. Idéal pour les animations fluides. Se rappelle de lui-même pour créer une boucle d'animation.

Savoir quand utiliser chacun. Pour une animation fluide : `requestAnimationFrame`. Pour un délai simple : `setTimeout`. Pour une répétition régulière : `setInterval`.

## Les erreurs fréquentes

Oublier d'écouter les événements. Le bouton existe, le code aussi, mais rien ne se passe au clic. Vous avez créé l'élément mais oublié d'ajouter `addEventListener`.

Sélectionner un élément qui n'existe pas. Votre `querySelector('.maClasse')` renvoie `null` parce qu'aucun élément n'a cette classe. Quand vous essayez de manipuler cet élément inexistant, tout plante.

Confondre `querySelector` et `querySelectorAll`. Le premier retourne un seul élément, le second une liste. Si vous utilisez `querySelector` sur une classe qui apparaît plusieurs fois, vous n'aurez que le premier élément.

Oublier de stopper `setInterval`. L'intervalle tourne indéfiniment même quand ce n'est plus nécessaire. Il faut le stopper avec `clearInterval` quand vous avez fini.

Mélanger CSS et JavaScript. Vous écrivez `element.style.background-color = 'red'` au lieu de `element.style.backgroundColor = 'red'`. Le tiret ne fonctionne pas en JavaScript, il faut la camelCase.

Copier du code sans le comprendre. Vous trouvez une solution sur Stack Overflow, vous la collez, ça marche. Mais vous ne savez pas pourquoi. À l'examen, vous ne saurez pas l'adapter.



## Ce qu'il faut faire

Écrivez du code simple et clair. Pas besoin de faire compliqué pour impressionner. Un code simple qui fonctionne vaut mieux qu'un code complexe qui plante.

Testez au fur et à mesure. N'écrivez pas cinquante lignes d'un coup. Écrivez une ligne, testez dans la console, vérifiez que ça marche, passez à la suivante.

Utilisez la console pour déboguer. Si quelque chose ne marche pas, ajoutez des `console.log()` pour voir ce qui se passe. Vérifiez que vos variables contiennent ce que vous pensez qu'elles contiennent.

Lisez les messages d'erreur. Ils vous disent souvent exactement ce qui ne va pas. "Cannot read property of null" signifie que vous essayez de manipuler un élément qui n'existe pas. "Uncaught TypeError" signifie généralement que vous appelez une méthode sur quelque chose qui n'est pas du bon type.

Commentez le code si nécessaire. Pas besoin de commenter chaque ligne, mais si la logique est complexe, expliquez ce que vous faites.



## Exemples de code

### Changer le texte d'un paragraphe au clic

```javascript
// Sélectionner le bouton et le paragraphe
const bouton = document.querySelector('#monBouton');
const paragraphe = document.querySelector('#monParagraphe');

// Écouter le clic sur le bouton
bouton.addEventListener('click', function() {
  paragraphe.textContent = 'Le texte a changé !';
});
```

### Ajouter une classe au survol

```javascript
// Sélectionner l'élément
const element = document.querySelector('.maDiv');

// Ajouter la classe au survol
element.addEventListener('mouseenter', function() {
  element.classList.add('survol');
});

// Retirer la classe quand la souris part
element.addEventListener('mouseleave', function() {
  element.classList.remove('survol');
});
```

### Faire apparaître un élément après un délai

```javascript
// Sélectionner l'élément (initialement caché en CSS)
const message = document.querySelector('#message');

// Le faire apparaître après 3 secondes
setTimeout(function() {
  message.classList.add('visible');
}, 3000);
```

### Animation simple avec requestAnimationFrame

```javascript
// Sélectionner l'élément à animer
const boite = document.querySelector('#boite');
let position = 0;

// Fonction d'animation
function animer() {
  position += 2; // Avancer de 2 pixels
  boite.style.left = position + 'px';

  // Continuer l'animation si on n'est pas arrivé à 500px
  if (position < 500) {
    requestAnimationFrame(animer);
  }
}

// Démarrer l'animation
animer();
```

### Créer et ajouter un nouvel élément

```javascript
// Créer un nouveau paragraphe
const nouveauParagraphe = document.createElement('p');
nouveauParagraphe.textContent = 'Ceci est un nouveau paragraphe';
nouveauParagraphe.classList.add('nouveau');

// L'ajouter à la fin du body
document.body.appendChild(nouveauParagraphe);
```

### Exemple combiné : bouton qui déclenche une animation

```javascript
const bouton = document.querySelector('#demarrer');
const boite = document.querySelector('#boite');

bouton.addEventListener('click', function() {
  // Ajouter une classe active au bouton
  bouton.classList.add('actif');

  // Changer le texte après 1 seconde
  setTimeout(function() {
    boite.textContent = 'Animation en cours...';
  }, 1000);

  // Ajouter une classe d'animation
  boite.classList.add('anime');
});
```
